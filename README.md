# PRODIGY_WD_03 – Tic Tac Toe Web Application

## 📌 Overview
This project is an interactive Tic Tac Toe web application built using HTML, CSS, and JavaScript.

It includes a multi-round match system (Best of 3, 5, 7 or custom rounds) with real-time score tracking and winner detection.

## ✨ Features
- Two-player gameplay (X vs O)
- Winner detection logic
- Draw detection
- Match system (Best of 3 / 5 / 7 / Custom)
- Score tracking
- Highlight winning combination
- Responsive and modern UI

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
